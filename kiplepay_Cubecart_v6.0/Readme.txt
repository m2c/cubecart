Descriptions:
============
This is a Kiplepay plugin for CubeCart Version 6

Note
=========================
Kiplepay supports only Mylaysian currency (MYR), so recommended not to use multiple currency.

Installation Steps:
=========================================
1. Upload the Kiplepay folder to the modules/gateway folder.
2. Navigate EXTENSIONS => Manage Extensions, Select Kiplepay to edit configuration and save the updates.
3. Select "Clear cache" the click the submit button.
